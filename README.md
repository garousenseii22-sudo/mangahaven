# MangaHaven

React + TypeScript + Tailwind + PWA starter for a clean manga reader.

## Run locally


1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open http://localhost:5173


## Notes
- Placeholder images are in `public/icons/`.
- API functions in `src/api/mangaAPI.ts` return static data for easy prototyping. Replace fetch calls as needed.
